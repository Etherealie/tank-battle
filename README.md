# 坦克大战

一个可直接打开游玩的单文件浏览器小游戏，使用原生 HTML、CSS 和 Canvas JavaScript 实现，无需安装依赖或运行构建命令。

## 开始游玩

双击 `index.html`，或用任意现代浏览器打开它。

操作方式：

- `WASD` / 方向键：移动和转向
- `Space`：开火
- `P`：暂停 / 继续
- `Enter`：开始、下一关或重新开始

移动端会显示触控方向键和开火按钮。

## 游戏规则

- 击毁本关所有敌军坦克即可进入下一关。
- 砖墙可以被子弹击碎，钢墙和水面不可通过。
- 敌军从战场上方三个入口增援，会主动移动和开火。
- 基地被击中或玩家生命归零都会失败。
